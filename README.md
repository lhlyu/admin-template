# admin-template

> vue3 + vite + typescript + pinia + vue-router

> 一个通用的admin管理项目模板，仅实现了简单的登录和菜单

## 特征

- ~~没有使用任何的ui框架~~可以跟任何vue3 UI框架搭配
- 根据路由自动生成菜单、面包屑、标签页

## 搭配
